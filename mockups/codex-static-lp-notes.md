# Codex static LP concept notes

## 作成物

- `mockups/codex-static-lp-concept.html`
- `mockups/codex-static-lp-concept.png`（1440 × 3581）

CodexにLP全体の静止画カンプ作成を依頼し、Codexが生成したHTMLをHermes側でPlaywrightレンダリングしてPNG化した。

## 良い方向性

- AIっぽいサイバー感を避け、白/クリーム/深紺/琥珀色でBtoB向けに落ち着いた印象になっている。
- ファーストビューの「配ったあとも、印象が残る小さな贈りもの。」は、飴そのものではなく利用価値を訴求できている。
- 用途別カード、制作工程、価格目安、CTAまで1枚で構成が見える。
- 既存LPより「法人ノベルティの実用品質」に寄っている。

## 実装へ移すべき改善点

1. ファーストビューはこの方向性に寄せる
   - 深紺の締め色
   - 和紙・紙箱・個包装の質感
   - CTAは「用途に合わせて相談する」「価格目安を見る」の2軸

2. 用途別導線を強化する
   - 展示会・商談会
   - 周年記念
   - 採用イベント
   - 店舗・ホテル

3. 価格セクションは「検討しやすさ」を出す
   - 20円〜/粒
   - 38円〜/粒
   - 780円〜/箱
   のような比較カードはLPにそのまま使いやすい。

4. 制作工程は不安解消セクションとして見せる
   - 用途と数量の整理
   - 色・ロゴ・包装設計
   - 校正と仕様確定
   - 製造・検品・納品

5. CTAは押し売りではなく「まず配布シーンだけ教えてください」にする
   - 法人案件の初回相談ハードルを下げる。

## 注意点

- 画像素材は既存assetsを使用しているため、最終的には実写/生成画像の品質を揃える必要がある。
- 文字量はやや多いので、実装時はスマホ表示で少し削る。
- このカンプはLP全体の方向性確認用。実装時は現在の`index.html`へ構造・配色・セクション設計を移植する。
