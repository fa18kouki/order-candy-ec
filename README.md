# AMEIRO — オーダーメイド飴ECサイト プロトタイプ

企業ノベルティ・結婚式・推し活・地域イベント向けの、オーダーメイド飴ECサイトの1ページプロトタイプです。

## 開き方

```bash
open /Users/kannokouki/dev/order-candy-ec/index.html
```

## 実装済みの画面要素

- ファーストビュー
- 商品ラインナップ3種
- デザインプレビュー
- 色・文字・味の簡易カスタマイズ
- 概算見積もりフォーム
- 用途別活用例
- FAQ
- レスポンシブ対応

## インタラクション

- ロゴ/文字入力で飴プレビューが更新される
- 味の選択でサブテキストが更新される
- 色スウォッチで飴の配色が変わる
- 商品タイプ・数量変更で概算見積もりが更新される
- フォーム送信で保存トーストが表示される

## 次に本番化するなら

1. 商品仕様を確定する（最小ロット、単価、印刷制約、納期）
2. 決済方式を決める（Shopify / Stripe / 請求書払い中心）
3. 入稿フローを決める（画像アップロード、AI/人力データ確認）
4. 製造元・在庫なし受注生産フローを確定する
5. SEO用に用途別LPを分ける（展示会ノベルティ、結婚式プチギフト、推し活など）
